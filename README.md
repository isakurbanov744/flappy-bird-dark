# Flappy Bird Dark
Dark version of Flappy Bird game made using Pygame in Python 3.8

**Requirements**

- PyGame
- Python 3.x.x

**Instructions**
- Run the main.py file from ./application/main.py
- Note: This project was made using Pycharm Community Edition 2019 - Version 3.4
